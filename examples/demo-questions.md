# Demo Questions

以下问题适合录屏或现场演示：

1. 这个仓库主要解决什么问题？
2. README 里怎么解释 graph-based agent orchestration？
3. 项目安装步骤是什么？
4. 文档里如何说明 multi-agent workflow？
5. 最近有哪些 issue 在讨论错误处理？
6. 最近有哪些 issue 提到 streaming 或 state 管理？
7. 最近的 PR 主要在改什么？
8. PR 评论里最常见的反馈点是什么？
9. 如果我问一个当前仓库没有覆盖的问题，系统会怎么处理？
10. 如果文档和讨论线程出现冲突，系统会怎么表达？
11. 哪些内容更适合从 README 获取，哪些更适合从 Issue / PR 获取？
12. 为什么这个系统适合做 Agent 能力展示，而不只是普通 RAG？
13. trace_id 能拿来做什么？
14. 这个系统如何保证回答可追溯？
15. 为什么模型接入要做成 OpenAI-compatible 抽象？
